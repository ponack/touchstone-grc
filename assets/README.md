## Mark description

Asymmetric flight feather formed by metallic assay streak marks (copper → bronze → gold → electrum → silver) on a dark basalt touchstone, inside a triple-ring hexagonal frame with circuit-node branches. Feather leans 15° off-vertical. Part of the **Forged in Feathers Technology** brand family — shares hex-frame language with Crucible IAP.
